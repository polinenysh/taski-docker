# Taski-Docker

Учет и планирование задач в контейнеризированном приложении с готовым API и CI/CD.

**Taski-Docker** — это приложение для управления задачами (создание, редактирование, пометка как выполненные). Оно объединяет:
	•	фронтенд (SPA-приложение на React),
	•	бэкенд (Django REST API),
	•	связку между фронтом и API,
	•	инфраструктуру Docker и Docker-Compose,
	•	автоматизацию CI/CD для сборки и деплоя.  ￼

Проект организован так, чтобы можно было запускать весь стек в контейнерах: frontend, backend, gateway/прокси-сервис, база данных и другие службы.

## Основные возможности
	•	Полный пользовательский интерфейс для списка задач.
	•	REST-API на Django для работы с данными задач.
	•	Локальный запуск через Docker, единый docker-compose.yml.
	•	Настроенные CI/CD workflows для автоматической сборки и тестирования.  ￼

### Как установить на локальный компьютер

1) Клонировать репозиторий

```
git clone https://github.com/polinenysh/taski-docker.git
cd taski-docker
```

2) Установить Docker и Docker-Compose

Убедитесь, что на вашей машине установлены:
	•	Docker (Engine),
	•	Docker Compose.

3) Запустить приложение в контейнерах

```
docker compose up --build
```

Эта команда соберёт образы и запустит все сервисы: backend, frontend, gateway и базу данных.

4) Проверить работу

После запуска приложение будет доступно по адресу: http://127.0.0.1:8000

5) Остановка
```
docker compose down
```
